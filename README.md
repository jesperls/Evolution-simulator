# Evolution-simulator
Tries to simulate evolution using NEAT

current inputs:
  1-16. Raycasts 16 rays around the agent which returns if the ray hit a wall, pellet or other agent(represented by the genetic distance between the two)
  17. Agent rotation in degrees
  18. Agent stamina
  19. Agent health
  20. Agent breeding cooldown

current outputs:
  1. Go forward
  2. Go backwards
  3. Turn left
  4. Turn right
